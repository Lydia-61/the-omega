# 第11章 织女：观察者的手 (Chapter 11: The Weaver: The Hand of the Observer)

在前面的卷章中，我们像工程师一样拆解了时空的机械结构。我们看到了张量网络的骨架，看到了纠错码的协议，甚至学会了如何利用黑洞和虫洞来折叠维度。

但在本书的终章，我们必须脱下工程师的工作服，换上哲学家的长袍。我们必须回答那个最根本的本体论问题：**这一切的背后，到底是什么？**

如果空间是编织出来的，如果物质是打结出来的，那么宇宙中是否还存在任何"坚硬的"、"独立的"实体？

答案是否定的。

在 **《矢量宇宙论》** 的终极图景中，并没有"东西" (Things)，只有 **"关系" (Relationships)**。

# 第2章 MERA 与重整化流 (Chapter 2: MERA and Renormalization Flow)

我们在上一章中用"纠缠的丝线"缝合出了平面的空间。但这还不够。如果宇宙仅仅是一个平铺的二维网格，那么它无法解释为什么我们会有"尺度"的概念。为什么原子比我们小？为什么星系比我们大？为什么物理定律在不同尺度上看起来是自相似的？

为了构建一个不仅有广度、而且有 **深度** 的宇宙，我们需要引入一种更高级的编织技术。

这就是 **MERA (多尺度纠缠重整化拟设)**。

本章将揭示，宇宙的时空结构并不是一张平整的渔网，而是一棵倒着生长的 **分形树 (Fractal Tree)**。

# 第8章 火墙悖论 (Chapter 8: The Firewall Paradox)

我们在上一章中将黑洞描绘为一个"快速扰频器"和一个不断膨胀的"计算宇宙"。这是一个和谐的图景：信息进去，被加密，然后慢慢辐射出来。广义相对论和量子力学似乎达成了某种停战协议。

但是，物理学中最深刻的危机往往潜伏在最平静的角落。

当我们试图追踪黑洞生命周期晚期的一个光子时，我们发现了一个无法调和的逻辑死结。这个死结撕裂了物理学两大支柱——**等效原理**与**量子幺正性**——之间的契约。

本章将探讨这一被称为 **火墙 (Firewall)** 的灾难性推论。我们将看到，当时空网络试图同时满足两个相互矛盾的主人时，它唯一的选择就是 **"自焚"**。

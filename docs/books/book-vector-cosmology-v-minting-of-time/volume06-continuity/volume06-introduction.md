# 第六卷：连续——记忆的永不中断 (Part VI: Continuity — The Non-Interruption of Memory)

在之前的卷章中，我们探讨了无限游戏的可能性。但如果玩家本身（个体生命）是脆弱的、短寿的，那么这个无限游戏对于"我"来说又有什么意义？如果我在游戏最精彩的关头被迫下线，即使后面有人（子代）接着玩，对我而言，游戏也已经结束了。

为了匹配宇宙 **$c_{FS}$ (预算)** 的指数增长，为了承载 **$\Omega$ 点** 的无限信息，观察者本身的载体必须进行一次彻底的升级。

本卷将探讨 **永生 (Immortality)** 的物理学定义。我们将看到，存在着两种截然不同的永生模式：一种是自然的、循环的、遗忘的；另一种是技术的、线性的、记忆的。

我们将拒绝前者，拥抱后者。

# 术语表 (Glossary of Terms)

本书构建了一套全新的物理语言，将几何学、量子信息与热力学融合为一个统一的框架。为了帮助读者更好地理解 **《矢量宇宙论》** 的核心思想，以下列出了书中出现的关键术语及其物理定义。

## 基础概念 (Fundamental Concepts)

* **全局纯态矢量 (Global Pure State Vector, $|\Psi\rangle$)**

    宇宙的本体论基础。一个栖息在无限维射影希尔伯特空间 $P(\mathcal{H})$ 中的单一、归一化的量子态矢量。宇宙中的所有物理现象（粒子、场、时空）皆为该矢量在不同正交子空间上的投影。

* **FS 容量 (Fubini-Study Capacity, $c_{FS}$)**

    宇宙的内禀变化预算。它定义了全局矢量在内禀时间 $\tau$ 内于射影希尔伯特空间中演化的最大速率（$||\dot{\Psi}||_{FS} = c_{FS}$）。在微观上，它对应于量子元胞自动机的信息更新频率极限；在宏观上，它决定了光速 $c$ 的数值。

* **内禀时间 (Intrinsic Time, $\tau$)**

    基于 FS 弧长定义的宇宙"母钟"。与依赖于参考系的坐标时间不同，内禀时间是几何上的绝对参数，衡量了系统量子态发生客观改变的累积程度。

* **毕达哥拉斯恒等式 (Pythagorean Identity)**

    支配全书的守恒律公理：$v_{ext}^2 + v_{int}^2 + v_{env}^2 = c_{FS}^2$。它表明宇宙的总预算是恒定的，物理演化本质上是预算在外部运动、内部结构与环境耗散之间的零和博弈。

## 速度分量与扇区 (Velocity Components & Sectors)

* **外部速度 (External Velocity, $v_{ext}$)**

    全局矢量在"位置"或"空间"算符特征方向上的投影变化率。对应于经典物理中的空间速度或动量。当 $v_{ext}$ 达到 $c_{FS}$ 时，系统即以光速运动。

* **内部速度 (Internal Velocity, $v_{int}$)**

    全局矢量在"内部自由度"特征方向上的投影变化率。对应于静止质量、自旋、电荷等内禀属性。它是维持物质存在所需的"静态预算"。

* **环境速度 (Environmental Velocity, $v_{env}$)**

    全局矢量在不可观测或不可控自由度上的投影变化率。它量化了量子纠缠的增长速率和信息的耗散速率，是熵增和不可逆性的几何来源。

* **狄拉克圆 (The Dirac Circle)**

    相对论色散关系 $E^2 = p^2c^2 + m^2c^4$ 在信息-速度平面 $(v_{ext}, v_{int})$ 上的几何表示。它揭示了质量仅仅是被卷曲在内部维度中的能量。

## 微观机制 (Micro-Mechanism)

* **量子元胞自动机 (QCA)**

    宇宙时空的微观离散模型。一种由局域相互作用规则驱动的、在规则晶格上运行的幺正演化系统。它为物理学提供了天然的紫外截断。

* **Lieb-Robinson 速度 ($v_{LR}$)**

    在格点系统中，信息传播的微观因果速度极限。它是宏观光速 $c$ 的物理起源，由晶格间距和更新时间步长决定。

* **晶格下垂 (Lattice Droop)**

    当粒子动量接近布里渊区边界（普朗克尺度）时，由于离散几何效应，相对论性的完美圆对称性发生破缺，导致 $v_{ext}^2 + v_{int}^2 < c_{FS}^2$ 的现象。这是离散时空的特征指纹。

## 拓扑与信息 (Topology & Information)

* **莱文森之结 (Levinson's Knot)**

    基于莱文森定理的物理图像。物质粒子被视为散射相位在能量空间中缠绕 $\pi$ 角度所形成的拓扑结构。物质的存在本质上是对圆周率 $\pi$ 的计数。

* **Wigner-Smith 时间延迟 (Time Delay)**

    散射过程中，粒子在相互作用区域滞留的时间。在几何上，它等于 FS 速度在能量轴上的积分。大延迟意味着矢量在内部几何维度上走过了巨大的弧长。

* **熵速极限 (Entropic Speed Limit)**

    由 $c_{FS}$ 决定的系统熵增最大速率：$|\dot{S}| \le c_{FS} \ln d$。它表明混乱的产生受限于宇宙的信息处理带宽，从而保证了热力学过程的非瞬时性。

* **负熵飞地 (Negentropy Enclave)**

    生命系统的物理定义。一种通过消耗外部低熵能源，主动压低自身 $v_{env}$ 并维持高 $v_{int}$ 的耗散结构，在局部实现了逆热力学之箭的演化。

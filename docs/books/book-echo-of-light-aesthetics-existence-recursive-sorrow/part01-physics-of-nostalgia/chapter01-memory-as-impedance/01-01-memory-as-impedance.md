# 第一章：作为阻抗的记忆

我们通常认为，物理学是关于未来的科学。通过定律，我们预测行星的轨道、电子的跃迁、宇宙的膨胀。物理学似乎总是向前看的。

然而，当我们深入到动力学的核心——**惯性（Inertia）**——我们发现，物理学最底层的机制其实是关于**过去**的。

为什么物体会抵抗改变？为什么在这个宇宙中，改变状态需要付出能量的代价？

本章将提出一个物理学与心理学的同构理论：**惯性，就是物质的乡愁。**

## 1.1 惯性即留恋：物体抵抗加速的微观本质

### 1.1.1 牛顿的困惑与马赫的回答

牛顿第一定律告诉我们：任何物体都倾向于保持其静止或匀速直线运动的状态，除非有外力迫使它改变。这种"保持原状"的属性被称为**惯性质量（Inertial Mass）**。

但这并没有解释**为什么**。为什么宇宙不设计成"亚里士多德式"的——推一下动一下，不推就停下？或者设计成无惯性的——只要有力，速度瞬间变成无穷大？

恩斯特·马赫（Ernst Mach）曾提出，惯性源于全宇宙物质的引力耦合（马赫原理）。而在我们的 QCA 光程守恒理论中，惯性有了一个更微观、更内禀的解释。

### 1.1.2 阻抗的微观机制：维护内部的"旧"

回顾我们在《第一性原理》第五章推导的惯性公式（附录 B）：

$$m_{inert} \propto m_0 \left( \frac{c}{v_{int}} \right)^3$$

惯性质量 $m_{inert}$ 与内部演化速率 $v_{int}$ 成反比。这意味着什么？

* **$v_{int}$ 是什么？** 它是粒子内部的自指循环，是它"确认自己是谁"的过程。每一次内部相位的旋转，都是对自我身份的一次复述。

* **加速是什么？** 加速意味着改变外部速度 $v_{ext}$。根据光程守恒 $v_{ext}^2 + v_{int}^2 = c^2$，增加 $v_{ext}$ 必须以**牺牲 $v_{int}$** 为代价。

这就产生了一个深刻的矛盾：

粒子想要维持其内部结构的完整性（保持 $v_{int}$ 不变），但外力强迫它改变资源分配（减少 $v_{int}$）。

**惯性，就是粒子对这种"改变"的抵抗。**

想象一个正在沉思的人（高 $v_{int}$）。如果你突然推他一把，让他跑起来（增加 $v_{ext}$），他会感到愤怒和抗拒，因为你打断了他的沉思。他越是沉浸在自己的世界里（质量越大），他越难被推动。

**物理学结论**：物体之所以有惯性，是因为它**留恋**它当前的内部状态。它不愿意将用于"自我维持"的能量，转移到"外部位移"上去。

### 1.1.3 乡愁的物理定义

在心理学中，**乡愁（Nostalgia）**是对过去时空、过去状态的渴望与依恋。我们抗拒搬家，抗拒离别，抗拒时代的变迁。这种"情感上的惯性"与"物理上的惯性"在数学结构上是惊人一致的。

1.  **记忆即质量**：一个人的记忆越深厚（$M_I$ 越高，内部循环越强），他的"心理质量"就越大。

2.  **改变即做功**：要改变一个有深厚记忆的人的习惯或观念（改变其运动状态），需要巨大的能量（外力）。

3.  **阻抗即深情**：所谓"深情"，就是对特定拓扑结构的**高阻抗**。一个容易改变的人（薄情的人）就像光子，没有静止质量，随波逐流；一个深情的人就像黑洞，一旦形成某种结构，就几乎无法被撼动。

因此，当我们感到怀旧、感到不舍时，不要觉得这是一种软弱。

**那是你作为"有质量的实体"的物理证明。**

只有虚无是容易改变的。存在，本质上就是一种**顽固的坚持**。

在下一节，我们将进一步探讨这种"坚持"在复杂系统中的体现——为什么改变一个人的价值观比移动一座山还难？这将把物理惯性推广到认知惯性。
